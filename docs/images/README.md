# Screenshots & diagrams

Place PNG/SVG assets referenced by the README and docs here, e.g.:

- `files.png` — Files explorer
- `ask.png` — Ask your files (AI)
- `audit.png` — Audit log
- `share.png` — Share dialog

Capture them from a locally running instance (see the root README → Quick Start).
Keep images optimised (< 500 KB where possible) and use the same theme for a
consistent gallery.
